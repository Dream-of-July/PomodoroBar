PomodoroBar 1.0 RC 2b

  • Adds automatic update checks powered by Sparkle.
  • Shows localized release notes before installing an update.
  • Downloads updates first, then asks before installing and relaunching.
  • Uses separate main and Legacy appcasts so each build only receives compatible updates.
  • Improves update packaging for faster app replacement.
  • Fixes the bottom rhythm selector mask so the selected pill is clipped by the rounded container correctly.
  • Fixes the updater state so the top update button can recover after the app relaunches or the release notes dialog is canceled.
  • Adds a throttled launch-time update check so fresh installs can discover updates without waiting for the daily Sparkle schedule.
  • Repairs future-dated Sparkle check times caused by manual system clock changes.
